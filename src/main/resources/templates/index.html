Java Spring приложение - интернет-магазин с упрощенным функционалом.

Предоставление доступа к различным частям сайта осуществляется
посредством авторизации на основе ролей USER / ADMIN.

Часть сайта для посетителей (Show all bicycles) - покупка товара.
В административной части (Admin panel) - наполнение сайта товаром,
просмотр поступивших заказов.

Новые пользователи создаются с ROLE_USER. Нет возможности создать
новых администраторов.

Есть 2 пользователя по умолчанию:

  • ad [ad] - администратор
  • sss [sss] - покупатель
